== Description ==
An AJAX Stars Rate Box with slide down effect and opinions collect system for your WordPress blog's page. To show off your StarsRateBox just put <code><?php starsratebox(); ?></code> in your template.

Appearance in main design themes is now corrected.

Live examples and skins preview you can find on our website.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php starsratebox(); ?>` in your templates
